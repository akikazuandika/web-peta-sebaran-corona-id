# Web peta sebaran corona di indonesia
## Technology
- Php
- Laravel
- Bootstrap
- Nodejs
- Npm
- Vue

## Requirements 
- Php
- Npm
- Nodejs

## Datasource
Kawalcorona (https://kawalcorona.com/api/)

## How to run
1. Clone this repository
2. Run this command
- ``` cd web-peta-sebaran-corona-id ```
- ``` npm i ```
- ``` php artisan serve ```
